Work Examples

Here are just a few of my Projects that I have completed overtime. Developing applications is a passion for myself, I love Computer Programming and I find that learning new languages is easier when you just dive on in with a project that you feel passionate about.

Edwards Rentals

Edwards Rentals was a simple freelance job for Apartment listings in Muncie. It is currently a work in progress. Development with the Google Maps API was utilized with this website in order to showcase the rentals and their locations.

Top Boat

A CS222 Project that involved the creation of a Javafx Application for playing the Classic Battleship with some twists. This is still a work-in-progress but so far I have had a very fun time working on it. Once it has been completed it will be added to a Public Github repository.

Birthday Bot

Birthday Bot is a Slack bot application written in PHP that announces birthdays for the Digital Corps. It also comes with slash commands that are able to provide users with the birthdays in a month or with a Corps member's birthday.
